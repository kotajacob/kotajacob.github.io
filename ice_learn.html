Ice Slide

It's a little puzzle game about pushing ice cubes in order to put out fires. It's still in heavy development and needs a lot of work but you can feel free to try out the beta's as they come out.

The game is open source so if you're tyring to learn Gamemaker studio with code then it might be a good place to explore. If you notice a bug or have a feature suggestion you can submit an issue on github or just email me at my normal email kotawalsh@gmail.com
